## Rich Text Editor {#editor}

Our advanced editor combines the simplicity of markdown with the power of rich formatting. Create complex documents with ease using our intuitive interface.

<CodeGroup tabs="Markdown,HTML">
  ````markdown
  # Document Title

  This is a **bold** statement with `inline code`.

  | Feature | Description |
  |---------|-------------|
  | Tables | Easy to create |
  | Code | Syntax highlighted |
  ````
  ````html
  <h1>Document Title</h1>
  <p>This is a <strong>bold</strong> statement with <code>inline code</code>.</p>
  <table>
    <tr><th>Feature</th><th>Description</th></tr>
  ````

## Integrations and Automation

Connect WIZIUM with your existing tools to create seamless workflows. Our API and webhooks enable automation and data synchronization.

<CodeGroup tabs="Webhook,API Call">
  ````json
  {
    "event": "document.updated",
    "document": {
      "id": "doc_123",
      "title": "API Reference",
      "author": "john.doe@company.com"
    }
  }
  ````
  ````python
import requests

# Update document via API
response = requests.patch(
    "https://api.wizium.com/v1/documents/doc_123",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={"title": "Updated API Reference"}
)
  ````
</CodeGroup>
