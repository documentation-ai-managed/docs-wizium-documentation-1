## Security and Compliance

Your documentation security is our top priority. WIZIUM employs industry-standard encryption for data at rest and in transit, along with role-based access controls to ensure only authorized team members can view or edit sensitive information.

<ExpandableGroup>
  <Expandable title="Encryption Standards" default-open="true">
    All data is encrypted using AES-256 encryption, and we support SSO integration with providers like Okta and Auth0.
  </Expandable>
  <Expandable title="Compliance Certifications" default-open="false">
    WIZIUM is SOC 2 Type II certified and GDPR compliant, making it suitable for enterprise use.
  </Expandable>
</ExpandableGroup>
